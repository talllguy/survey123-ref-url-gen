Survey123 URL Generator

A conversion script to generate Survey123 URLs for web maps

See the Esri Survey123 blog post on the Survey123 custom URL scheme for more information.

By Elliott Plack and GitHub contributors. Made with ♥ in Baltimore

Enter the Survey123 item ID. Find it by opening the survey on survey123.arcgis.com and copying the ID in the URL.
Enter the PK field name that Collector uses. It will be populated in the Survey123 form to maintain referential integrity. Separate multiple keys by comma (e.g. PK1,PK2,PK3).
Enter the FK field name that Survey123 uses. It should receive the primary key from Collector referenced above. Separate multiple keys by comma (e.g. FK1,FK2,FK3).
Enter the text for the button.
Put this HTML in the web map pop-up HTML area to display a survey button with the generated URL link.
